mtx.baseObjects.SolidObject¶

class SolidObject(id, symbol)[source]¶

Bases: mtx.BaseObject.BaseObject

A solid object can not be moved, removed or collected. No other object can be moved to a cell where a solid object is placed.

IsSolid()[source]¶
Returns:True if the object is solid, False otherwise.
