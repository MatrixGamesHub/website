mtx.Field¶

class Field(level, width, height)[source]¶

Bases: object

Class to manage a two-dimensional game field. The field consists of rows and columns, and each intersection is represented by a cell.

Parameters:
  • width (int) – The width of the field.
  • height (int) – The height of the field.
Clear()[source]¶
